# 起重机械选型系统 — 重构规格文档
> 基于用户描述的参数体系与两种工况场景

## 一、参数字母体系（替换对照表）

| 旧字母 | 新字母 | 含义 | 单位 |
|--------|--------|------|------|
| G | G | 构件重量（额定起重量需求） | t |
| R | R | 吊装半径（回转中心→构件中心水平距离） | m |
| Hb（建筑总高）| Hj | 建筑物顶点标高（建筑总高） | m |
| Hc（机械高度）| h | 吊臂铰点距地面高度（机械高度） | m |
| d（结构距离）| D | 吊车最外缘→建筑外表面水平距离 | m |
| sH（构件安装高度）| Ha | 构件安装位置标高（地面以上） | m |
| α_min（碰撞仰角）| — | 废弃（由系统自动计算）| ° |
| — | Lz | 主臂臂长（用户输入/数据库查询） | m |
| — | Lf | 副臂臂长（用户输入/数据库查询） | m |
| — | α | 主臂与水平线夹角 | ° |
| — | β | 副臂安装角度（相对水平面）| ° |
| — | Hd | 吊臂顶端距离地面高度（计算输出） | m |
| — | d_boom | 建筑物顶点至吊臂的垂直距离（计算值） | m |
| q | q | 安全系数（默认 1.1）| — |
| H_crane | h | 同上（机械高度）| m |

> **注**：建筑总高 Hj 与构件安装高度 Ha 是两个不同概念：
> - Hj = 建筑物总高度（地面→建筑顶）
> - Ha = 构件安装位置标高（地面→安装点）

## 二、系统整体架构

### 2.1 选型流程

```
用户输入参数
    │
    ▼
┌─────────────────────────┐
│  自动判断工况场景         │
│  ① 有 Hj+D 输入 → 场景B  │
│  ② 无 Hj+D 输入 → 场景A  │
└────────┬────────────────┘
         │
    ┌────┴────────────────┐
    ▼                    ▼
 场景A                场景B
 无建筑物阻碍          有建筑物阻碍
    │                    │
    ▼                    ▼
 载荷表查询            载荷表查询
 + 几何约束           + 几何约束
 + 碰撞检查           + 碰撞检查
    │                    │
    ▼                    ▼
 结果展示               结果展示
（单一示意图）         （工况类型对应示意图）
```

### 2.2 工况类型（来自载荷表）

| 载荷表 key | 工况名称 | 说明 |
|-----------|---------|------|
| `main_boom` / `主臂工况` | 主臂工况 | 纯主臂，无副臂 |
| `主副臂工况` | 主副臂工况 | 主臂+副臂（延伸） |
| `jib_standard` | 塔式副臂工况 | 固定角度塔式副臂 |

## 三、场景A — 无建筑物阻碍

### 3.1 输入参数

| 参数 | 字母 | 说明 | 备注 |
|------|------|------|------|
| 构件重量 | G(t) | 必填 | — |
| 吊装半径 | R(m) | 必填 | — |
| 主臂臂长 | Lz(m) | 必填 | 来自载荷表可选值 |
| 副臂臂长 | Lf(m) | 选填 | 不填=无副臂 |
| 副臂安装角度 | β(°) | 条件必填 | 有副臂时必填 |
| 主臂角度 | α(°) | 必填 | — |
| 构件安装高度 | Ha(m) | 选填 | 影响安全检查 |
| 安全系数 | q | 默认 1.1 | — |
| 机械高度 | h(m) | 默认 1.5 | 吊臂铰点高度 |

### 3.2 无副臂计算逻辑

**几何公式：**
- `R = Lz × cos(α)` → 用于验算输入是否一致
- `Hd = Lz × sin(α) + h` （吊臂顶端高度）
- `h_req = Ha + 1` （吊钩最低高度要求）

**选择条件：**
- `Hd ≥ h_req + 1.5` （构件距吊臂顶端 ≥ 1.5m 安全净空）
- `Q ≥ G × q` （载荷表额定起重量 ≥ 需求）
- 效率 = `G / Q` （越高越好）

### 3.3 有副臂计算逻辑

**几何公式：**
- 吊臂总长 = `Lz + Lf`
- 副臂顶端高度需根据 β 和 α 联合计算
- 碰撞检查同无副臂（无建筑物，故只需性能满足）

**选择条件：** 同无副臂

### 3.4 工况判断规则

系统自动根据以下顺序尝试工况：
1. 主臂工况（载荷表中有 `main_boom`）
2. 主副臂工况（载荷表中有 `主副臂工况`，且 Lf > 0）
3. 塔式副臂工况（载荷表中有 `jib_standard`，α 为固定值）

## 四、场景B — 有建筑物阻碍

### 4.1 输入参数（在场景A基础上增加）

| 额外参数 | 字母 | 说明 |
|---------|------|------|
| 建筑物总高 | Hj(m) | 建筑顶点标高 |
| 结构距离 | D(m) | 吊车最外缘→建筑水平距离 |
| 吊臂顶端与建筑顶垂直距离 | d_boom(m) | **系统计算输出**，默认 ≥ 1m |

### 4.2 无副臂计算逻辑

**几何公式：**
- `R = Lz × cos(α)`
- `Hd = Lz × sin(α) + h`
- `d_boom = Hd - Hj` （系统计算）
- `α_min = atan((Hj - h) / D)` （最小安全仰角）
- `R_actual = Lz × cos(α)`

**选择条件：**
- `α ≥ α_min` （仰角满足最小安全要求）
- `d_boom ≥ 1` （吊臂不撞建筑，默认值 1m，可调）
- `Q ≥ G × q` （载荷满足）
- `Hd ≥ Ha + 1.5` （构件净空）

### 4.3 主副臂联合计算逻辑

**适用条件：** 无副臂方案不满足碰撞约束时

**几何公式：**
- 已知：R, Hj, D, h, Lz, Lf, α, β
- 校核：吊臂顶端是否越过建筑物顶点
- 副臂顶端点 = 吊臂顶端 + Lf 在角度 β 方向上的延伸
- `d_boom = Hd_jib - Hj` （副臂顶端与建筑顶垂直距离）

**选择条件：**
- `d_boom ≥ 1`
- `Q ≥ G × q`

### 4.4 塔式副臂工况

**特点：** α 由机械固定（不用户输入），β = {0°, 15°, 30°} 等固定档位

**计算：**
- 系统枚举所有可行的 (Lz, β) 组合
- 计算 Hd 和 d_boom
- 判断是否满足 `d_boom ≥ 1` 和 `Q ≥ G × q`

### 5.3 公式面板（右侧，responsive 隐藏）

**场景A — 无建筑物阻碍：**
- `R = Lz × cos(α)`
- `Hd = Lz × sin(α) + h`
- `h_req = Ha + 1` （吊钩最低高度）
- `Q ≥ G × q` （载荷约束）
- `Hd ≥ h_req + 1.5` （安全净空）

**场景B — 有建筑物阻碍：**
- `R = Lz × cos(α)`
- `Hd = Lz × sin(α) + h`
- `d_boom = Hd - Hj`
- `α_min = atan((Hj - h) / D)`
- `d_boom ≥ 1` （安全距离）
- `α ≥ α_min`

**场景B + 副臂：**
- `R_jib = Lz×cos(α) + Lf×cos(β)`
- `Hd_jib = Hd + Lf×sin(β)`
- `d_boom = Hd_jib - Hj`

## 六、UI 布局设计

### 6.1 选型页面结构

```
┌──────────────────────────────────────────────────────┐
│  标题栏：起重机械选型                                  │
├────────────────────┬─────────────────────────────────┤
│                    │                                  │
```

### 6.2 场景切换逻辑

- 选择"无建筑物阻碍"：隐藏 Hj、D 输入框，示意图切换为简化版（无建筑）
- 选择"有建筑物阻碍"：显示 Hj、D 输入框，示意图切换为完整版（有建筑）
- 副臂勾选：切换主臂/主副臂两种载荷表查询
- 塔式工况：自动识别（jib_standard 载荷表存在时启用）

## 七、输出结果设计

### 7.1 结果展示字段

| 字段 | 说明 |
|------|------|
| 推荐机型 | 最优机型（综合效率最高）|
| 额定起重量 Q | 该工况+臂长+半径下的额定载荷 |
| 效率 | G / Q（越高越好）|
| 臂长 | 推荐臂长 Lz |
| 仰角 α | 实际使用角度 |
| Hd | 吊臂顶端高度 |
| d_boom | 碰撞安全余量（场景B）|
| 工况类型 | 主臂工况 / 主副臂工况 / 塔式副臂 |
| 不满足原因 | 如不满足，列出具体原因 |

### 7.2 多臂长对比分析

当同一机械有多个臂长可选时，表格展开显示：
- 臂长 1：Q=XX, 效率=XX%, d_boom=XX
- 臂长 2：Q=XX, 效率=XX%, d_boom=XX
- ...

## 八、实现优先级

**Phase 1：** 场景A（无建筑物阻碍）+ 主臂工况 + 结果展示
**Phase 2：** 场景B（有建筑物阻碍）+ 碰撞检查 + 示意图切换
**Phase 3：** 副臂工况 + 塔式工况 + 多臂长对比分析
